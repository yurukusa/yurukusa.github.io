1 · What changes

The change in one paragraph

Before June 15, a Claude Code subscriber running claude -p automation drew from the same usage pool as their interactive session. After June 15, the Agent SDK path draws from a separate monthly dollar credit, sized by plan. Beyond the credit, calls bill at standard API rates (if extra-usage is enabled) or stop (if it is not). Source: Anthropic support article 15036540.
