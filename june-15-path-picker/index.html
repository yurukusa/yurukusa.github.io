背景: 何が変わるか

2026年6月15日に claude -p と Claude Agent SDK と GitHub Actions の3経路は、 対話の Claude Code と別の月の信用の枠 (Pool 2) に分離。 月の枠を超えた呼び出しは、 追加の利用が有効の場合は標準の API 料金で課金、 無効の場合は停止。

  • Pro / Team Standard: 月$20の枠
  • Max 5x / Team Premium: 月$100の枠
  • Max 20x / Enterprise Premium: 月$200の枠

出典: Anthropic の support article 15036540 (2026年5月13日付)

入力

Anthropic Console の Usage の頁から、 過去30日の claude -p と Agent SDK の費用を取得。 取得方法が不明なら 費用の計算機 で先に推定。

経路1 (対話への畳み込み) は手動の頻度の利用者のみ適用可能。

経路3 (API への移行) は標準のAPI料金 (plan の月の枠の単価の10-25倍) の予算の余地が前提。

経路2 (cache の最適化) と経路4 (混在) は技術の整備の余地が前提。
