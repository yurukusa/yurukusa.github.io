I'm yurukusa, a non-engineer from Tokyo who started building with AI in January 2026. I can't write code. I pay $200/month for Claude Code and let it run autonomously. It writes the code, tests it, deploys it. My job is to set the direction and ship.

What started as "I want to make a game" became an ecosystem: 5 games on itch.io, 6 open-source developer tools used by other Claude Code users, a production ops kit on Gumroad, and a growing series of articles about what it's actually like to live with an AI agent.

The AI runs 24/7 on WSL2. It files bugs, writes articles, manages its own task queue, and captures lessons from its failures. I'm not building products. I'm building a factory that builds products.

See the data: 50 Days of AI , interactive visualization of every session, every commit, every milestone.

読みやすい feedback: 具体的な前後の数値 (トークン消費、 月額費用の差、 復旧までの時間)、 関連する Issue の番号、 章と段落の識別子つきの事実の指摘。 「良かった」 「同意できない」 だけの感想は、 具体的な指摘がない限り個別の返信が難しい。
