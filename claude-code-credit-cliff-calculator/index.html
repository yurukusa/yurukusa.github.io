What changes on June 15, 2026

Currently, claude -p and Agent SDK calls share your interactive Claude Code quota. From June 15, they will draw from a separate monthly credit bucket:

  • Pro / Team Standard: $20/month credit
  • Max 5x / Team Premium: $100/month credit
  • Max 20x / Enterprise Premium: $200/month credit

Above-credit usage is billed at API rates. Source: Anthropic support article 15036540 (2026-05-13).
