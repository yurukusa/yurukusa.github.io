6月15日に何が変わるか

現在、 claude -p と Agent SDK の呼び出しは、 対話の Claude Code と同じ枠を使う。 6月15日からは、 別の月の credit の枠が分離される:

  • Pro / Team Standard: 月$20の credit
  • Max 5x / Team Premium: 月$100の credit
  • Max 20x / Enterprise Premium: 月$200の credit

credit を超えた分は API の単価で課金される。 出典: Anthropic の support article 15036540 (2026年5月13日付)。

入力

入力の費用が何日分の利用かを書く。

Anthropic の Console の利用の頁から、 非対話の呼び出しの分を取得。

対話の Claude Code の中で /usage --json の指令から取得、 または Anthropic の Console の利用の頁から API の呼び出しごとに分解して見積もる。

Agent Viewの倍率の補足 (2026年5月16日に追加): Claude Code Agent View の Background Sessions(2026年5月の版2.1.139以降)を使うと、 並列のagentの各々が全速で利用枠を消費する。 10件の並列のbackground agentで、 1件の対話型のセッションの約10倍の速度で枠を焼く。 加えて、 各セッションの隣の1行のsummaryは Haiku の模型で生成され、 通常のAPI課金で実行される。 上記の見積もりに、 実際の並列のagentの数を掛けて判定する。 Agent View の4件の罠の操作者の側の整理の解説。
