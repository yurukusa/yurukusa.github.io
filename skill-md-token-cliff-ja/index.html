SKILL.md Token Cliff

CLAUDE.md を貼り付ける。 1 セッション毎の token の数、 plan に応じた月の費用の見立て、 5 件の浪費の型 (禁止の一覧 / 反復の例 / 冗長の説明 / 多段の手順 / 強調の積み重ね) の検出と、 各々の修正の手順が出る。

2026-05-16 更新。 token の見立ては OpenAI の BPE の概算 (英語は 4 文字で 1 token、 日本語は 2 文字で 1 token)。 月の費用は 2026-04-22 以降の Anthropic の価格と、 CLAUDE.md が毎ターン読み込まれる前提の見立て。 40% の節約の数値は、 30,000 閲覧超の Qiita 記事で記録された 1 利用者の Skills への移行の事例の上限の値。

1. CLAUDE.md (または SKILL.md、 毎セッション読み込まれる任意の参照ファイル) の中身を貼る
中身は外部に送信されない。 全ての解析はブラウザの中で動く。 自分の本物の設定で試した方が、 検出の意味が出る。
2. 利用中の plan は?
月の費用の見立てに使う。
3. 1 日 の平均の turn の数 は?
「turn」 は Claude との 1 往復のこと。 CLAUDE.md は毎ターン読み込まれるため、 turn の数で費用が乗算する。

token を食う 5 件 の 型
数値は見立てで、 実測ではない。 実際の費用は plan の cache の挙動、 Sonnet と Opus の使う割合、 CLAUDE.md が system prompt として読み込まれているか会話の中に入っているか (cache の当たり方が違う) で大きく変わる。 Anthropic の prompt cache の有効期間は 5 分で、 5 分以上の無作業の後は cache が外れて全額の課金となる。 型の検出は概算で、 詳細な点検は商品の本で扱う。 中身はブラウザの外には出ない。
